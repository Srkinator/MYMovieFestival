# Movie Festival

Movie Festival is a simple app writen in vanilla JavaScript where you can add movies, programs, movies to programs and create a Festivals.

[DEMO](https://srkinator.github.io/MyMovieFestival/)


# Showcasing


-   Vanilla JS
-   Prototypes
-   Form Validation


# Stack

-   ES5 + ES6
-   Semantic UI
